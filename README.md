# tailscale-gnome-qs

##### INSTALL
```bash
sudo tailscale set --operator=$USER
git clone https://github.com/joaophi/tailscale-gnome-qs.git ~/.local/share/gnome-shell/extensions/tailscale@joaophi.github.com
```
